* Add Elementor/Gutenberg Template under a menu.
* Individual Menu Width Control Option.
* Option to add Sub Menu Position.
* General Option & Admin Panel for Controls.
* Create Multiple Menu Page.
* Build Menu Content by Elementor or Gutenberg.
* Option to Setup Menu.
